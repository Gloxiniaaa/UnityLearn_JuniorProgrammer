Brief summary
-------------

This pathway teaches how to make games with Unity using C# and the Unity Editor. Key topics covered:

- C# scripting fundamentals and best practices
- Unity Editor workflow and project structure
- Player input and the Input System
- Physics and collision handling
- Basic animation and Animator workflows
- UI creation and event handling
- Debugging, iteration, and small playable projects

What you'll find in this repository
----------------------------------

- The Unity project files used to complete the pathway (in the `Assets` and `ProjectSettings` folders)
- A demo video of the project located at `Public/Demo.mp4`
